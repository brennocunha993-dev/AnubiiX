Perguntas Frequentes

Encontre respostas rápidas para as dúvidas mais comuns dos nossos jogadores

Como entrar no servidor AnubiiX?

Para entrar no servidor, siga estes passos:

1. Abra o Minecraft (versões 1.16 a 1.20)

2. Clique em "Multiplayer"

3. Clique em "Add Server"

4. Digite o IP: play.anubiix.com

5. Clique em "Done" e depois "Join Server"

Quais versões do Minecraft são suportadas?

O servidor AnubiiX suporta versões do Minecraft de 1.16 a 1.20.

Recomendamos usar a versão 1.19.4 para a melhor experiência de jogo, pois é a versão otimizada do nosso servidor.

Como proteger minha construção?

Para proteger sua área, use o sistema de claims:

1. Pegue uma pá de ouro

2. Clique nos dois cantos opostos da área que deseja proteger

3. Use /claim para confirmar a proteção

4. Use /trust [jogador] para dar permissão a outros jogadores

Para mais comandos, digite /help claim

Como ganhar dinheiro no servidor?

Existem várias formas de ganhar dinheiro no AnubiiX:

• Loja do servidor: Venda itens usando /shop

• Missões: Complete quests com /quests

• Trabalhos: Use /jobs para escolher uma profissão

• Eventos: Participe dos eventos diários e semanais

• PvP: Ganhe recompensas nas arenas de combate

Como criar ou entrar em um clã?

Sistema de clãs do AnubiiX:

• Criar clã: /clan create [nome]

• Entrar em clã: /clan join [nome]

• Convidar jogador: /clan invite [jogador]

• Ver informações: /clan info

• Chat do clã: /clan chat [mensagem]

Como reportar um jogador?

Para reportar um jogador que está quebrando as regras:

1. Use /report [jogador] [motivo] no jogo

2. Ou abra um ticket aqui no site

3. Ou reporte no nosso Discord

Importante: Tenha evidências (screenshots/vídeos) do comportamento inadequado.

Como apelar uma punição?

Se você foi punido injustamente:

1. Abra um ticket de apelação nesta página

2. Forneça seu nome de usuário e detalhes da punição

3. Explique por que acredita que a punição foi injusta

4. Nossa equipe analisará seu caso em até 48 horas

Nota: Apelações falsas podem resultar em punições adicionais.
